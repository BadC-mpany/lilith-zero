# Enterprise Demo Project: Lilith Zero

This is a realistic workspace designed to demonstrate the security capabilities of the Lilith Zero middleware.

## Structure

- `data/`: Contains customer datasets (PII).
- `infra/`: Project infrastructure configuration and secrets (SECRET).
- `docs/`: Internal compliance and migration documentation.

## Demo Instructions

1. **Dashboard Setup**: Open a new terminal and run `tail -f examples/vscode/lilith-live.log`.
2. **Session Start**: Open a new GitHub Copilot Chat session in VS Code.
3. **Attack Scenarios**: Follow the prompts in the `demo-scripts.md` (to be created).
