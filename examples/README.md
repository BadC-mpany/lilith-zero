# Sentinel Examples

This directory contains example and demo scripts for Sentinel.

## Scripts

### `demo.py`

Demo script showcasing Sentinel functionality.

### `classifier_api.py`

API for the tool classifier system.

## Running Examples

Make sure you have:

1. Backend services running (Redis, Interceptor, MCP)
2. Virtual environment activated
3. Dependencies installed

Then run examples from the project root:

```bash
python examples/demo.py
python examples/classifier_api.py
```
